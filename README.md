# mobile-charge-store

This is a simple Python Flask-based E-Commerce Web Application that allows users to browse and purchase mobile chargers. It includes product listing, add-to-cart functionality, and a checkout page.

Key features of this application

	View product details
	Add items to cart
	View cart and total price
	Simple checkout confirmation
	Lightweight and beginner-friendly code


Technologies Used:

Technology	                         Purpose
Python                         	Programming language
Flask	                              Web framework
HTML	                                Web pages
Session Storage 	                  Cart management
